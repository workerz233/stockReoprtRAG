# Eval Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为项目新增一套可运行的离线 RAG 评测入口，自动生成样本并输出检索与回答质量指标。

**Architecture:** 在 `evals/` 目录下新增两个独立单元：样本生成器负责从现有项目 Markdown 构造评测数据集，评测执行器负责加载数据集并调用真实 `ResearchRAGPipeline`。评分保持确定性，避免引入第二个评审模型。

**Tech Stack:** Python 3, pathlib/json, 现有 FastAPI 项目模块, unittest
